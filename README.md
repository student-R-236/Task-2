# Task 2 – Phishing Email Analysis

## Objective
Analyze a suspicious email and identify phishing indicators such as spoofed sender address, misleading content, urgent language, and domain inconsistencies.

## Files Included
- sample-email.txt — The phishing email used for analysis.
- Phishing-Analysis-Report.md — Detailed breakdown of phishing indicators.
- README.md — Overview of the task.

## Summary
The email pretending to be from the Microsoft Account Team was analyzed. Multiple red flags were identified, including a fake sender domain (msupdate.net), urgent language, generic greeting, lack of Microsoft branding, and suspicious instructions. These indicators classify the email as a phishing attempt.
